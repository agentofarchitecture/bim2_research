## Signal quality rubric

Each source should be scored quickly:

- **Strategic relevance:** Does it affect Rhino Mind / Agents of Architecture?
- **Technical maturity:** Can we build/use it now?
- **Adoption likelihood:** Will practitioners actually use this?
- **Differentiation:** Is this common knowledge or an edge?
- **Actionability:** Does it imply an experiment, feature, partnership, or note?
